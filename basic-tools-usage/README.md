# 기본적인 도구들의 사용방법

이 문서는 대학 생활 중 유용하게 활용할 수 있는 필수 도구들에 대한 설명을 모아놓은 것입니다. 여기에 열거된 도구들을 잘 활용하면 컴퓨터를 이용한 반복 작업을 줄이고, 정보를 효율적으로 얻고, 정리하며 사용할 수 있게 될 것입니다.

인트로덕션에서 말 하였듯이,

1. 컴퓨터 공학도라면 알았으면 좋겠는 것
2. 대학생이라면 알았으면 좋겠는 것



이 두 가지에 대해서 이야기를 할 예정입니다.

즉, 대학생활에서 쓰게 될

1. 읽을거리 관리 도구
   1. 메일
   2. RSS 리더
   3. 논문 관리 도구
2. 개발 도구
   1. 터미널
   2. 에디터
   3. 패키지 관리자
   4. FTP
   5. ssh/telnet
   6. 버전 관리 도구
3. 협업 도구
   1. Slack
   2. 일정 관리도구
   3. 문서 작성 도구
   4. 클라우드 동기화 서비스
   5. 쉬운 버전 관리 도구
   6. 이슈 트래커
4. 삶에 도움이 되는 도구
   1. 패스워드 관리 도구
   2. 슬라이드 쉐어
5. 애플 교도라면 써볼 만한 것들
   1. 정말
   2. 많음

에 대해서 이야기를 해보려고 합니다.

이 섹션에서는 다룰 내용이 많지만, 차근차근 익혀나가면 큰 도움이 될 것입니다.
